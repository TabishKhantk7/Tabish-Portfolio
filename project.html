Selux App

Case Study iPad App

Overview

This healthcare application was expertly designed to help employees seamlessly sell specialized hearing aids on demand. The core function involves an intuitive user flow seamlessly guiding through audiometric tests, product selection, and checkout.


The Challenge

Translating complex healthcare terminology into a very simple, digestible path for both the operator and the patient was the priority. I utilized clean typography, bright accent colors, and generous white space to create a calming and professional aesthetic.
